## 🎯 プロダクト到達目標

D-Aikeiba は、文字や数字の羅列しかない競馬情報を、各ファクタで評価基準を設けて全頭見える化し、市場に騙されない本当の馬の力をはかるシステム。

**KPI 必達ライン**（詳細: memory/project_kpi_targets.md）

| 自信度 | 的中率 | 単勝回収率 |
|---|---|---|
| SS | 60.0% | 150.0% |
| S | 50.0% | 120.0% |
| A | 40.0% | 100.0% |
| B | 25.0% | 90.0% |

## 🏗️ アーキテクチャ全景

```
[netkeiba/JRA/NAR] → scraper → SQLite DB → engine/ML → pred.json → dashboard (Flask + React) → Cloudflare Tunnel
```

詳細は `memory/architecture.md` / `memory/architecture_frontend.md`。

---

## 📋 制約・前提

- **netkeiba レート制限**: time.sleep() でレート制限。並列リクエスト禁止 (詳細: feedback_netkeiba_concurrent_throttle)
- **SQLite WAL モード**: 読み取り並列安全、書き込みシリアル
- **MLモデル**: ~2GB メモリ。ProcessPoolExecutor 非推奨
- **データ取得優先順位**: 公式 (JRA/NAR) → 競馬ブック → netkeiba
