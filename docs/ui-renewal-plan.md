# UIリニューアル設計案 — 競馬ブック風タブレイアウト

## コンセプト

競馬ブックのレース詳細ページのように、**1レースに対して複数の切り口（タブ）**で情報を閲覧できるUI。
現行の「1画面に全情報を縦積み」から「タブで切り替えて各情報を深く見る」構造に変更。

---

## タブ構成（11タブ）

競馬ブックの既存タブ10種 ＋ D-AIオリジナルの「展開」タブ。

| # | タブ名 | 競馬ブック対応 | データソース |
|---|--------|-------------|-------------|
| 1 | **出馬表** | syutuba | `/api/race_prediction` (horses) |
| 2 | **オッズ** | odds | `/api/race_prediction` + `/api/predicted_odds` |
| 3 | **AI予想** | cpu（相当） | `/api/race_prediction` (composite, marks, formation) |
| 4 | **レース結果** | seiseki | 外部リンク or 新API（結果取得済み時） |
| 5 | **調教** | cyokyo | `/api/race_prediction` (training_records) |
| 6 | **能力表** | nouryoku | `/api/race_prediction` (past_runs, ability_grades) |
| 7 | **厩舎の話** | danwa | `/api/race_prediction` (horse_comment, horse_diagnosis) |
| 8 | **コースデータ** | data | `/api/venue/bias` + `/api/db/course_stats` |
| 9 | **血統表** | kettou | `/api/race_prediction` (sire_grade, mgs_grade, bloodline) |
| 10 | **当該成績** | tougaiseiseki | `/api/race_prediction` (past_runs フィルタ) |
| 11 | **展開** | なし（D-AIオリジナル）| `/api/race_prediction` (pace, formation) |

## Tab 1: 出馬表（デフォルト表示）

競馬ブックの syutuba ページ相当。コンパクトな全馬一覧。

```
┌──┬──┬──┬──┬───────────┬──────┬────┬──────┬────┬────┬────┬────┬────────┐
│D印│印│枠│番│馬名       │騎手  │斤量│ｵｯｽﾞ  │総合│能力│展開│適性│前走     │
├──┼──┼──┼──┼───────────┼──────┼────┼──────┼────┼────┼────┼────┼────────┤
│◎│◎│5 │9 │ディープXX  │ﾙﾒｰﾙ │58 │ 2.3  │ 68│ 65│ 70│ 62│①① 1着 │
│  │○│3 │5 │タイトルYY  │川田  │57 │ 4.8  │ 63│ 60│ 58│ 65│③② 2着 │
│  │▲│7 │13│スピードZZ  │横山武│56 │ 8.2  │ 59│ 62│ 55│ 58│⑤④ 3着 │
│  │△│1 │2 │ワンダーAA  │松山  │56 │12.5  │ 56│ 55│ 60│ 54│②③ 5着 │
│  │★│8 │16│ミラクルBB  │岩田望│54 │35.0  │ 52│ 48│ 58│ 55│⑧⑦ 4着 │
└──┴──┴──┴──┴───────────┴──────┴────┴──────┴────┴────┴────┴────┴────────┘
```

**カラム:**
- D印: ユーザー自印（既存DMarkCell流用）
- 印: AI印（◉◎○▲△★☆×）色付き
- 枠: 枠色背景
- 番: 馬番
- 馬名: 太字 + 性齢小文字（例: 牡4）
- 騎手: 短縮名
- 斤量: kg
- オッズ: 単勝（色分け: 1-3倍=赤太字, 3-10倍=黒, 10倍以上=灰）
- 総合/能力/展開/適性: 偏差値（セル背景=5色グレード）
- 前走: 通過順+着順（コンパクト）

**行クリック → 既存の馬詳細展開（アコーディオン）を維持**

## 技術判断

| 項目 | 判断 | 理由 |
|-----|------|------|
| タブ実装 | `useState` + 条件レンダリング | 11タブ程度なら軽量実装で十分 |
| タブ切替時のデータ | 全タブ同一API応答を共有 | `/api/race_prediction` の1回取得で9タブ分のデータを持つ |
| Lazy Loading | Tab 8 (CourseData) のみ | 追加APIコール必要なタブだけ遅延 |
| URL連動 | `#tab=odds` 形式 | ブックマーク・共有対応 |
