# course_db_preload の各項目への影響一覧

1レース分析（`--race_id`）で事前収集DB（約77,000走）をマージした場合、各分析項目へどの程度影響するかを一覧化した。

---

## 凡例

| 記号 | 意味 |
|------|------|
| ◎大 | 直接かつ大きく影響する |
| ○中 | 影響するが、一部のみ or 条件付き |
| △小 | 間接的にわずかに影響する |
| — | 影響なし（course_db を使用しない） |

---

## 一覧表

| カテゴリ | 項目 | 影響度 | 説明 |
|----------|------|--------|------|
| **A-E章 能力** | 基準タイム | ◎大 | StandardTimeCalculator が course_db から同コース1〜3着の平均を算出。サンプル数で精度・信頼度が変わる |
| | 基準タイム信頼度 | ◎大 | 30走以上→A / 10以上→B / 3以上→C。preload でほとんどのコースが A に |
| | 走破偏差値 | ◎大 | 基準タイムとの差を換算定数で偏差値化。基準タイム精度に直結 |
| | 換算定数 | ◎大 | course_db が 30走以上あると自動校正。サンプル増で換算定数が安定 |
| | 能力偏差値 (max/WA/α) | ◎大 | 上記すべての結果を集約した能力偏差値 |
| | 着差プロファイル (E-3) | — | 馬の過去走のみ使用。course_db 非依存 |
| | トレンド (E-1) | — | 馬の過去走の偏差値推移から判定。course_db 非依存 |
| **F章 展開** | ペース予測 (HH/HM/MM/MS/SS) | — | 出走馬の脚質・過去走・コース形状のみ。course_db 非依存 |
| | ペース信頼度 | — | 逃げ馬頭数などから判定。course_db 非依存 |
| | 基準上がり3F | ◎大 | pace_last3f_db（course_db から構築）でコース×ペース別の基準を算出。preload でサンプル充実 |
| | 推定上がり3F | ◎大 | 基準上がり3F + 末脚タイプ補正 + ペース補正 |
| | 展開偏差値 (base_score) | ◎大 | (基準上がり3F − 推定上がり3F) − 位置取り秒差 を偏差値化 |
| | 展開偏差値 (last3f/position/gate等) | — | 末脚評価・位置取り・枠順バイアス等は course_db 非依存 |
| | 前半3F・後半3F推定 | — | estimate_pace_times_from_runs は出走馬の過去走のみ使用 |
| | 展開コメント・有利脚質 | — | ペース予測と前半/後半3F推定に依存。course_db 非依存 |
| **G章 コース適性** | コース適性偏差値 | — | 馬の過去走・CourseMaster・騎手コース成績。course_db 非依存 |
| | 枠順バイアス | — | コース形状・頭数・芝/ダートの傾向。course_db 非依存 |
| | コース脚質バイアス (G-3) | — | course_style_stats_db から取得。現状は空のため 0.0 固定 |
| **H-J章 騎手・厩舎** | 騎手展開影響 | — | 騎手の過去成績。course_db 非依存 |
| | 騎手コース影響 | — | 騎手のコース別成績。course_db 非依存 |
| | 厩舎評価・調教・勝負気配 | — | 馬・騎手・厩舎データ。course_db 非依存 |
| **総合・買い目** | 総合偏差値 | ◎大 | 能力55% + 展開30% + コース15%。能力・展開の変化で変動 |
| | 勝率・連対率・複勝率 | △小 | 総合偏差値から算出。間接影響 |
| | 穴馬・危険馬スコア | △小 | 総合偏差値・オッズ・人気から算出。間接影響 |
| | 買い目生成 | △小 | 総合偏差値・期待値・自信度から。間接影響 |
| | 自信度・資金配分 | — | ペース信頼度・評価分布。course_db 非依存 |
| **その他** | 改修後コースフィルタ | △小 | course_db をフィルタ。データ増で該当コースのサンプルが充実し精度向上 |

## 影響経路の図解

```
course_db (preload マージ後)
    ├─→ StandardTimeCalculator
    │       ├─→ 基準タイム ────────────→ 能力偏差値
    │       └─→ 基準信頼度 ────────────→ 能力信頼度
    │
    ├─→ Last3FDBBuilder → pace_last3f_db
    │       └─→ Last3FEvaluator
    │               ├─→ 基準上がり3F ───→ 展開偏差値 (base_score)
    │               └─→ 推定上がり3F ───→ 展開偏差値
    │
    └─→ calibrate_conversion_constant (30走以上で)
            └─→ 換算定数 ──────────────→ 走破偏差値 → 能力偏差値
```

---

## まとめ

- **◎大 の影響**: 能力偏差値（基準タイム・換算定数・信頼度）、展開偏差値（基準/推定上がり3F）、総合偏差値
- **△小 の影響**: 勝率・穴馬・買い目、改修後フィルタ
- **影響なし**: ペース予測、前半/後半3F推定、コース適性、枠順バイアス、騎手・厩舎評価、自信度

preload を導入した効果は、主に**能力評価**と**展開評価**の精度向上に現れる。
