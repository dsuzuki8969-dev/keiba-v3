# クイックスタートガイド

**5分で競馬解析マスターシステムを動かす手順**

---

## ステップ1: インストール（2分）

### Windows
1. `setup.bat` をダブルクリック
2. 自動的にPythonパッケージがインストールされます
3. 完了したら何かキーを押す

### Mac/Linux
```bash
bash setup.sh
```

---

## ステップ2: 認証情報の設定（2分）

```bash
python -m src.setup_credentials
```

以下を順番に入力:

```
netkeibaのログインID: your_id
netkeibaのパスワード: your_password
競馬ブックSmartのログインID: your_id
競馬ブックSmartのパスワード: your_password
```

**注意:**
- netkeibaとKeibabook両方のアカウントが必要です
- パスワードは `~/.keiba_credentials.json` に保存されます（平文）

---

## ステップ3: デモ実行（1分）

```bash
python demo.py
```

`output/keiba_demo.html` が生成されます。ブラウザで開いて確認してください。

---

## ステップ4: 実際のレース分析（実践）

### 4-1: レースIDを取得
netkeibaで分析したいレースのURLを開く:
```
https://race.netkeiba.com/race/result.html?race_id=202506021011
                                                    ^^^^^^^^^^^^
                                                    これがレースID
```

### 4-2: 分析実行
```bash
python analyze.py
```

レースIDを入力:
```
レースIDを入力してください: 202506021011
```

### 4-3: 結果確認
`output/` フォルダに、レースIDとレース名を含むHTML（例: `202506021011_東京11R.html`）が生成されます。

---

## HTML出力の読み方（重要）

### 1. レース概要
- **展開予測**: ペースタイプ（MM=ミドル、HH=ハイなど）
- **信頼度**: A/B/C（展開の読みやすさ）

### 2. 全馬一覧
- **印**: ◎本命、○対抗、▲単穴、△連下、☆大穴
- **総合偏差値**: 高いほど強い（50が平均）

### 3. 印・穴馬・危険な人気馬
- **穴馬**: 実力があるのに人気がない馬
- **危険な人気馬**: 人気があるのに実力がない馬

### 4. 買い目
- **◎買い**: 期待値120%以上（積極的に買う）
- **○買い**: 期待値100-120%（検討の余地あり）
- **×買わない**: 期待値100%未満（見送り）

## トラブルシューティング

### エラー: ModuleNotFoundError
```bash
pip install -r requirements.txt
```

### エラー: 認証失敗
1. netkeibaとKeibabook Smartにログインできるか確認
2. `~/.keiba_credentials.json` を削除して再設定
3. パスワードに特殊文字が含まれている場合は注意

### HTMLが生成されない
1. レースIDが正しいか確認（12桁の数字）
2. オッズが確定しているレースか確認
3. ターミナルのエラーメッセージを確認
